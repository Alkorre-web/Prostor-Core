🗄️ База данных (SQLite)

Файл: database.py
Основная функция: get_db_connection()

Особенности:
- Используется sqlite3.Row для доступа по имени колонок
- Включены foreign keys (PRAGMA foreign_keys = ON)
- Автоматическое создание таблиц при запуске

Основные таблицы:
• users — пользователи (id, username, total_xp, level)
• skills — навыки с иерархией (user_id, parent_id, level, xp)
• tasks — задачи (user_id, skill_id, task_type, xp_reward)
• goals — цели (user_id, skill_id, title, description)
• subgoals — подцели (goal_id, position, is_completed)
• tags — теги (user_id, name, color)
• task_tags — связь задач и тегов (task_id, tag_id)
• xp_log — лог получения опыта (user_id, skill_id, xp_amount)

Пример запроса:
conn = get_db_connection()
skills = conn.execute(
    'SELECT * FROM skills WHERE user_id = ?',
    (user_id,)
).fetchall()
conn.close()
