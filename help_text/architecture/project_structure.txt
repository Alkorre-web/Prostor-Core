📂 Структура проекта

prostor_core.db — база данных SQLite

main.py — основной файл Flask приложения
├─ Инициализация БД: init_db()
├─ Авто-заполнение: add_test_data()
├─ Маршруты (routes):
│  ├─ /profile — страница профиля
│  ├─ /tasks — список задач
│  ├─ /goals — цели и подцели
│  ├─ /skills — дерево навыков
│  └─ /help — эта страница
└─ API endpoints:
   ├─ /api/skills/<id>/add-xp
   └─ /api/tasks/<id>/complete

database.py — работа с БД
├─ init_db() — создание таблиц
├─ get_db_connection() — подключение
└─ SAFE_MODE — флаг защиты от удаления

data_test.py — тестовые данные
└─ add_test_data() — заполнение БД при первом запуске

templates/ — HTML шаблоны (Jinja2)
├─ base.html — базовый шаблон с навигацией
├─ profile.html — профиль пользователя
├─ tasks.html — задачи
├─ goals.html — цели (двухпанельный вид)
└─ skills.html — дерево навыков

static/ — статические файлы
├─ css/main.css — основные стили
└─ js/main.js — JavaScript
